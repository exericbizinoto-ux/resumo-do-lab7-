# 📘 Resumo do Lab - DIO

Este repositório contém o resumo dos conhecimentos adquiridos durante o desenvolvimento do laboratório da DIO.

## 🚀 O que aprendi

Durante este lab, pude compreender melhor diversos conceitos importantes relacionados à computação em nuvem e ao Microsoft Azure.

### ☁️ Computação em Nuvem
Aprendi que a computação em nuvem permite o acesso a recursos como servidores, armazenamento e bancos de dados pela internet, sem a necessidade de infraestrutura física local.

### 🔐 Segurança e Identidade
Entendi a importância de serviços como:
- Azure Active Directory (agora Microsoft Entra ID) para gerenciamento de identidades
- Autenticação Multifator (MFA), que aumenta a segurança usando múltiplas formas de verificação
- Controle de acesso baseado em funções (RBAC)

### 🛡️ Governança e Conformidade
Conheci ferramentas importantes como:
- Azure Policy: usado para criar regras e garantir conformidade nos recursos
- Microsoft Purview: para governança e gerenciamento de dados

### ⚙️ Serviços do Azure
Aprendi sobre alguns dos principais serviços:
- Azure Virtual Machines: máquinas virtuais na nuvem
- Azure Functions: execução de código sem servidor (serverless)
- Azure Blob Storage: armazenamento de arquivos
- Azure Virtual Network: redes privadas na nuvem

### 🔎 Monitoramento
Entendi como funciona o monitoramento com:
- Azure Monitor
- Microsoft Defender for Cloud

## 📚 Conclusão

Este lab foi fundamental para entender os conceitos básicos do Microsoft Azure e da computação em nuvem, além de mostrar a importância da segurança, governança e gerenciamento de recursos.
